Unofficial Kaonavi Library for .NET

## Install

```powershell
# Package Manager
> Install-Package Kaonavi.NET

# .NET CLI
> dotnet add package Kaonavi.NET
```

## For Contributor
### Setup

```powershell
> dotnet tool restore
> dotnet restore
```

### Development Commands

```powershell
# Lint & Fix
> dotnet format

# Unit Test
> dotnet test

# Build
> dotnet build
```
